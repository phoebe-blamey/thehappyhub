🎨 PeaBe icon exporter

This page renders /icons/peabe.svg at four sizes and lets you save each as PNG. Drop the files into /public/icons/ and the iOS home-screen icon + Android PWA install will use them automatically.

What to do with these files

  1. Click each download button (or "Download all four").
  2. Move the four PNGs into the project's public/icons/ folder.
  3. Commit + push: the manifest + apple-touch-icon already reference these filenames, so they'll activate on next deploy.

Tip: if the rendered preview looks blurry, your SVG fonts may not have loaded yet. Refresh the page once and re-download.
