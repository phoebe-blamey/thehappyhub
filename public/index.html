Settings & Setup

Configure integrations and test your automations

🔌 Calendly webhook

When a client books the 90-Day Business Builder, a client file is auto-created with all intake answers. Add this URL in Calendly → Integrations → Webhooks → invitee.created:

https://hub.phoebeblamey.com.au/api/calendly-webhook

🎬 Zoom setup

For automatic transcript fetching, enable cloud recording + audio transcript in Zoom Settings → Recording. Then create a Server-to-Server OAuth app at marketplace.zoom.us and add ZOOM_ACCOUNT_ID, ZOOM_CLIENT_ID, ZOOM_CLIENT_SECRET to your Netlify environment variables.

📼 Zoom recordings — last 30 days

Recordings that match a client's name in the topic show as ✓ Matched. Anything else can be assigned manually below — pulls the transcript and saves it to that client.

💾 Backup all data — download a full snapshot

Belt-and-braces insurance. Downloads a single JSON file with EVERYTHING: every client, every program, every session note, plan, win, task, check-in, badge, plus your cohorts, plan templates, and activity log. Re-import isn't built yet, but if anything ever goes sideways with the server, this file is enough to rebuild from. Recommend running before each major sync or change.

🔄 Run full sync — Calendly + Zoom + dedupe

One-click: pulls every Calendly booking from the past year, every Zoom recording from the past year, creates client records for any that don't exist yet, attaches recordings to existing clients where it can, then merges any duplicates by email.

Use Dry run first to see what would change. Real run takes 1-3 minutes depending on volume.

📧 Email sending (Gmail API — live)

          
Outbound emails from the platform send via the Gmail API as phoebe@phoebeblamey.com.au and appear in your Gmail Sent folder automatically.

          
Configured via Netlify env vars: GOOGLE_CLIENT_ID, GOOGLE_CLIENT_SECRET, GOOGLE_REFRESH_TOKEN, GOOGLE_SENDER_EMAIL. To rotate credentials, see CLAUDE.md.

        


        

          
💬 SMS setup (Twilio)

          
To send SMS texts to clients, add three variables to Netlify: TWILIO_ACCOUNT_SID, TWILIO_AUTH_TOKEN, TWILIO_FROM_NUMBER

          
Get these from twilio.com → Console → Account Info. Cost is ~$0.01/SMS.

        


        

          
🔄 GitHub auto-deploy (never lose work)

          
Connect your Netlify site to a GitHub repo for automatic deploys. Every time a new version is built, push to GitHub → Netlify deploys automatically. No more zip uploads.

          
Netlify → Site settings → Build & deploy → Link to Git repository

❓ Quick help

      

        How do I track a win?
        
Open any of your programs and scroll to 🏆 Your wins, then tap "+ Log a win". Phoebe celebrates with you on her side too.

      

      

        Where can I see my plan?
        
Click the 📚 My Programs tab and pick the program you're working on. Your plan appears once you and Phoebe have completed your first session.

      

      

        How do I update my access code?
        
Open the ⚙️ Settings tab and tap Change code. You'll need your current code to confirm it's you.

      

      

        What if I forget my code?
        
Email Phoebe (button above) and she'll reset it for you. Easy.

      

      

        What does Phoebe see when I write something?
        
Anything you type in your portal — workbook answers, reflections, your notes, ticked tasks — Phoebe sees on her side. It\'s a shared space. The only thing she keeps private from you are her own personal session prep notes.
